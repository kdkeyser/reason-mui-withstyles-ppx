# Reason Mui PPX Generator

This library creates a code extension for conveniently using the `withStyles` functionality of MaterialUi in a typesafe way.

## Prerequisites

1. Install opam.

2. Install JS dependencies:

       yarn install

## How to (re)generate the ppx
    yarn clean
    yarn build

## Development
    yarn watch
